## 🔴 發現的實際問題

### 1. 架構違反單一職責

```
tts_engine.py 包含：
- ErrorHandler (錯誤處理)
- TextProcessor (文字處理)
- AsyncSynthesizer (非同步合成)
- PresentationTTS (主引擎)
```

**應該是**：4 個獨立檔案
**實際是**：1 個檔案

### 2. 音訊合併完全缺失

```python
# v557-new 的 synthesize_batch 只產生：
chunk_000.mp3
chunk_001.mp3
...

# 沒有任何合併邏輯！
# 不會產生最終輸出檔案
```

### 3. 熔斷器是假的

```python
class ErrorHandler:
    circuit_breaker_count = 0  # 類別變數！所有實例共用
    
    @staticmethod
    def should_circuit_break() -> bool:
        # 沒有狀態機
        # 沒有 recovery_timeout
        # 只是簡單計數
```

### 4. asyncio 會 crash

```python
# 在已有 event loop 的環境會 crash
loop = asyncio.new_event_loop()  # ❌ 錯誤
```

### 5. 整合測試不存在

```
text_processor.py → 404
retry_handler.py → 404
test_integration.py → 404
```
